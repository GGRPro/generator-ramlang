# generator-ramlang

> [Yeoman](http://yeoman.io) generator


## Getting Started

If you don't have yeoman then you will need to run this

```
$ npm install -g yo
```

To install generator-ramlang from npm, run:

```
$ npm install -g generator-ramlang
```

Finally, initiate the generator:

```
$ yo ramlang
```
